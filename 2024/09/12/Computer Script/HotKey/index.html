Win 快捷键


常用的 Windows 快捷键，按功能分类整理，便于查阅和使用：

常用基础快捷键

  • Win + D：显示桌面或恢复窗口。
  • Win + E：打开文件资源管理器。
  • Win + R：打开运行窗口。
  • Win + V: 打开粘贴面板
  • Alt + Tab：切换打开的应用程序。
  • Alt + F4：关闭当前窗口或退出程序。
  • Ctrl + C / Ctrl + V：复制 / 粘贴。
  • Ctrl + X：剪切。
  • Ctrl + Z / Ctrl + Y：撤销 / 恢复操作。
  • Ctrl + A：全选。
  • Ctrl + S：保存。
  • Ctrl + P：打印。

窗口管理

  • Win + ↑ / ↓：最大化 / 最小化窗口。
  • Win + ← / →：将窗口固定到屏幕左侧 / 右侧。
  • Win + Shift + ← / →：将窗口移动到左侧或右侧显示器（多显示器）。
  • Win + Tab：打开任务视图。

系统功能

  • Win + I：打开设置。
  • Win + L：锁定屏幕。
  • Win + Shift + S：启动屏幕截图工具。
  • Win + P：切换显示模式（扩展屏幕/投影等）。
  • Win + X：打开高级用户菜单（快速访问控制面板、任务管理器等）。
  • Win + Pause：打开系统属性窗口。

搜索与应用

  • Win + S：打开搜索框。
  • Win + Q：打开 Cortana（在支持的系统中）。
  • Win + 数字键：快速打开任务栏上的对应应用程序（从左到右）。

虚拟桌面

  • Win + Ctrl + D：新建一个虚拟桌面。
  • Win + Ctrl + → / ←：切换虚拟桌面。
  • Win + Ctrl + F4：关闭当前虚拟桌面。

文件管理

  • Ctrl + N：在文件资源管理器中新建窗口。
  • Ctrl + Shift + N：新建文件夹。
  • F2：重命名文件或文件夹。
  • Shift + Delete：永久删除（跳过回收站）。

辅助功能

  • **Win + + / -**：放大镜工具（缩放）。
  • Win + U：打开辅助功能设置。
  • Win + H：激活语音输入功能（在支持的系统中）。

命令提示符

  • Ctrl + C：中止当前命令执行。
  • Ctrl + A：选中所有命令提示符内容。
  • Alt + Enter：全屏/退出全屏命令提示符窗口。

运行命令


常用的 运行命令，可以在 Windows 的 Win + R 窗口中直接输入，快速打开常用工具或功能：

系统工具

  • **cmd**：打开命令提示符（CMD）。
  • **powershell**：打开 PowerShell。
  • **dxdiag**：打开 DirectX 诊断工具，查看系统和显卡信息。
  • **resmon**：打开资源监视器。

文件夹与位置

  • **shell:startup**：打开启动项文件夹。
  • **shell:Recyclebinfolder**：打开回收站。

常用程序

  • **notepad**：打开记事本。
  • **calc**：打开计算器。
  • **mspaint**：打开画图工具。
  • **snippingtool**：打开截图工具（部分版本为 Snip & Sketch）。
  • **mstsc**：打开远程桌面连接工具。
  • **control printers**：打开打印机和扫描仪窗口。
  • **osk**：打开屏幕键盘。

高级工具

  • **gpedit.msc**：打开本地组策略编辑器（专业版及以上可用）。
  • **secpol.msc**：打开本地安全策略。
  • **lusrmgr.msc**：打开本地用户和组管理器。
  • **diskmgmt.msc**：打开磁盘管理工具。
  • **compmgmt.msc**：打开计算机管理工具。

命令行工具


  • ipconfig：获取当前计算机的 IP 地址、子网掩码、默认网关
  • **msinfo32**：检查计算机硬件信息
